# Meeting Room Booking

> A Meeting Room Booking app demo using React, TypeScript, Redux, React Router, Material-ui and Jest

 To install Run 
 
 npm install(it may take a while depending on network speed)

 To run

 npm start 

 To build

 npm run build

 To run test

 npm run test
